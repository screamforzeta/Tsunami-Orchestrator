Steps to follow:
    1) Select the scan mode
    2) Select the folders to view/use during 
    Orchestrator execution:

        a. Tsunami Outputs: contains the logs 
        (not parsed) generated by Tsunami Scanner
    
    3) Enter the number of containers 
    to run simultaneously

        NB: containers use a large amount 
        of memory, if this is not sufficient, 
        the container will fail
        
        NB: the number of simultaneous containers must 
        not exceed the number of cores of the machine
    
    4) Click the "Start" button to start the scan
